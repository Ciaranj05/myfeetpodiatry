# My Feet Podiatry

Premium Next.js website for My Feet Podiatry, a foot health and movement clinic in Ballymena.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui primitives
- Framer Motion
- Lucide icons

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Checks

```bash
npm run lint
npm run build
```
